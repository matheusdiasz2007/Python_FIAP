## 📖 Sobre o Projeto

Este projeto consiste em uma agenda de contatos desenvolvida em Python utilizando estruturas de dados como:

- Dicionários
- Listas
- Funções
- Manipulação de arquivos
- JSON
- Entrada e saída de dados

O sistema permite cadastrar, editar, excluir e exportar contatos diretamente pelo terminal.

---

## ⚙️ Funcionalidades

✅ Adicionar contatos

✅ Adicionar múltiplos telefones e e-mails

✅ Alterar nome de contatos

✅ Editar formas de contato

✅ Excluir contatos

✅ Exportar agenda para TXT

✅ Exportar agenda para JSON

✅ Importar agenda via JSON

✅ Exibição organizada dos dados

## 🚀 Como executar

Clone o projeto:

```bash
git clone https://github.com/matheusdiasz2007/Python_FIAP.git
```

Entre na pasta:

```bash
cd Python_FIAP
```

Execute:

```bash
python manipulação_agenda.py
```
